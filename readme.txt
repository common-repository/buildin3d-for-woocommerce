Show 3D Animated Models as a product image in your woocommerce product in just 30 seconds.

== Description ==

А [live demo](http://buildin3d.com/product/fabbrix-jungle-life/).

Using our WordPress plugin “BuildIn3D for WooCommerce” you can change the product picture of your product and visualize an animated 3D model. This increases user engagement and conversion rates. Read more about what is the [added value when using 3D on your e-commerce website.](https://buildin3d.com/blog/what-is-the-added-value-of-3d-for-your-ecommerce-site)

Find out about [different integrations.](https://buildin3d.com/integrations/)

See [all avaliable models](https://platform.buildin3d.com/instructions) you can embed on your website.

How to use the plugin:

1. Go to the edit product page.
2. Scroll down to the **"BuildIn3d"** panel.
3. Visit the [gorilla instruction](https://platform.buildin3d.com/instructions/198-fabbrix-jungle-life-gorilla-in-3d-building-instructions)
4. Click on **Embed**.
5. Fill in the form and register.
6. Copy the **WooCommerce Embed Code**
7. Paste the code in the **B3 Embed Code** field
8. You should have the gorilla embeded on your product.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
